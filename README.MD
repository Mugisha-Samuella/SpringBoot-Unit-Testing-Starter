# Spring Boot Unit Testing Starter

A comprehensive starter project demonstrating various unit testing approaches in a Spring Boot application.

## Features

- JUnit 5 for unit testing
- MockMvc for controller testing
- TestContainers for integration testing
- @DataJpaTest for repository layer testing
- @WebMvcTest for controller layer testing
- @SpringBootTest for full context testing
- Test coverage with JaCoCo

## Prerequisites

- Java 17 or higher
- Maven 3.6.3 or higher
- Docker (for TestContainers)
